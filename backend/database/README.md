# Database Setup

Import [mubuga_tss.sql](/abs/c:/xampp/htdocs/MUBUGA-TSS/database/mubuga_tss.sql) into MySQL to create the starter database for the website.

Example with XAMPP MySQL:

```powershell
C:\xampp\mysql\bin\mysql.exe -u root < database\mubuga_tss.sql
```

The schema includes tables for:
- settings
- users
- pages
- programs
- staff
- news
- gallery
- messages
- admissions
- downloads
